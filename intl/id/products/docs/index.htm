Fitur premium

Selesaikan lebih banyak hal dengan Gemini di Dokumen

Buat dokumen profesional dari beberapa perintah sederhana dan dapatkan bantuan untuk memperbaiki konten Anda. Lihat ringkasan terbaru pekerjaan Anda dan ajukan pertanyaan untuk memperbaiki serta menyelesaikan dokumen Anda. Coba Gemini untuk Google Workspace sekarang juga.

 Membuat dokumen dengan mudah

Tidak perlu memulai dokumen dari awal - mulai dengan template dan elemen penyusun untuk mendapatkan tampilan profesional sejak awal.

Mulai dokumen Anda dengan kanvas cerdas

Buat email berformat, undangan kalender, pelacak ulasan, catatan rapat, dan lainnya secara instan di Dokumen dengan mengetik ‘@’.

Isi dokumen Anda dengan detail lengkap secara mudah

Gunakan chip smart untuk menambahkan nama, file, acara kalender, menu dropdown, dan lain-lain dengan mengetik ‘@’.

Hapus batas halaman dan menulislah tanpa batas

Pilih mode tanpa halaman untuk membuat konten - terus menulis dan men-scroll tanpa gangguan.

Tetap teratur dengan tab dokumen

Buat tab untuk setiap bagian project, rencana, atau laporan dan simpan semuanya di satu tempat - Anda tidak akan pernah kehilangan dokumen Anda lagi.

Berkolaborasi dan memperbaiki dokumen Anda, dari mana saja

Tetap sinkron dengan pengeditan dan komentar dari tim internal serta eksternal di ponsel, tablet, atau browser web Anda.

Berdiskusi langsung di dokumen

Fitur premium

Fitur ini tersedia di paket Google Workspace Business dan Enterprise.

Lakukan kolaborasi sambil melihat konteks melalui panggilan Meet di Google Dokumen, Spreadsheet, atau Slide.

Kontrol akses ke dokumen Anda

Izin yang mudah dikelola memungkinkan Anda mengontrol secara terperinci siapa saja yang dapat mengedit, mengomentari, mendownload, membagikan, atau melihat dokumen Anda.

Berkolaborasilah dengan partner Anda dan buat keputusan, semuanya di satu tempat

Gunakan komentar dan emoji untuk memberi masukan kepada tim dan tetapkan tugas untuk menjaga kelangsungan project tanpa berpindah dari dokumen.

Tinjau perubahan secara fleksibel

Anda dan para pemangku kepentingan dapat merujuk dokumen versi sebelumnya kapan saja dan mengaktifkannya kembali.

Mempercepat tugas sehari-hari

Buat catatan rapat dengan mudah dan cepat

Gunakan elemen penyusun catatan rapat di Dokumen untuk membuat draf catatan rapat yang terstruktur dan berkualitas dengan detail acara, langsung dari acara Google Kalender.

Sederhanakan alur kerja Anda dengan tanda tangan elektronik

Buat draf perjanjian, minta tanda tangan, dan kelola template kontrak, semuanya di satu tempat dengan Tanda tangan elektronik di Dokumen.

Hemat waktu dengan menambahkan konten yang sering Anda gunakan kembali secara mudah

Buat elemen penyusun kustom untuk potongan teks seperti biografi, tanda tangan, alamat, dan sebagainya - cukup ketik ‘@’ untuk memasukkan konten daripada menyalin dan menempelkan konten yang sama berulang kali.

Tetap aman

Data Anda dienkripsi secara default dan serangkaian upaya pengamanan – seperti enkripsi sisi klien canggih, langkah anti-penyalahgunaan, serta kontrol privasi – menjaga kerahasiaan data Anda.

Gunakan Dokumen untuk mempercanggih pengeditan jenis file yang populer

Impor jenis file populer seperti dokumen Microsoft Word dan file PDF, lalu kerjakan di Dokumen. Mengimpor file ke Dokumen membuka akses ke berbagai fitur asistif dan kolaboratif, termasuk komentar, item tindakan, dan kecerdasan bawaan.

Ingin tahu tentang Google Dokumen?

Baca FAQ kami untuk mengetahui informasi selengkapnya.

  • Siapa saja yang memiliki Akun Google dapat menggunakan Dokumen.

    Namun, beberapa fitur seperti bergabung ke panggilan Meet dari dokumen hanya tersedia di paket Google Workspace, dan fitur Gemini untuk Workspace seperti Bantu atur tersedia sebagai add-on. Lihat paket dan harga untuk organisasi atau Google Workspace Individual.

  • Anda dapat mengonversi dokumen Microsoft Word ke format Google Dokumen, atau Anda dapat secara langsung mengedit dokumen Word, tanpa harus membuat salinan. File asli akan tetap utuh.

  • Pengeditan bersama memungkinkan beberapa orang mengedit satu dokumen secara bersamaan, tanpa perlu bolak-balik mengirim salinan. Setelan berbagi memungkinkan Anda mengontrol siapa saja yang dapat melihat serta mengedit dokumen, dan histori revisi memungkinkan Anda untuk kembali ke versi sebelumnya.
