Часто задаваемые вопросы

  • В консоли администратора Google Workspace можно управлять доступом к приложениям, устанавливать их для определенных групп в организации, а также задавать настройки доступа этих приложений к данным пользователей.

  • Приложения Google Workspace предоставляются бесплатно. Чтобы использовать сторонние приложения, вам может потребоваться приобрести у их разработчиков дополнительные лицензии. Google не взимает плату за использование таких приложений. Собственные приложения также можно использовать бесплатно. Исключение составляют решения, созданные с помощью AppSheet, для доступа к которым пользователям требуется лицензия.

  • AppSheet – это интегрированный с Google Workspace сервис, который позволяет разрабатывать приложения без написания кода. С его помощью пользователи могут без труда создавать решения для оптимизации работы, просто перетаскивая модули, – даже если у них нет опыта программирования.

  • При разработке приложений дополнительная плата за лицензию Google Workspace не взимается. Вы можете использовать API и SDK Workspace бесплатно.

  • Вы можете ознакомиться с документацией для разработчиков Google Workspace, в том числе с информацией об Apps Script, на сайте https://developers.google.com/workspace. Чтобы узнать больше, подпишитесь на новостную рассылку, канал YouTube, сообщество для разработчиков или форум на сайте Stack Overflow.

  • Нет. В Apps Script доступны предварительно настроенные функции, такие как улучшенная конфигурация аутентификации, библиотеки кода и встроенный редактор кода. Это упрощает процесс разработки, но при создании приложений для Google Workspace не обязательно пользоваться этой платформой. Разработчики могут использовать любые инструменты и языки.

  • Разработчики могут использовать любые инструменты и языки. Создавать приложения для Google Workspace также можно с помощью Apps Script. На этой платформе доступны функции, упрощающие разработку, например улучшенная конфигурация аутентификации, библиотеки кода и встроенный редактор кода.
